VR Anatomy Viewer by Team OcuSonix | Scrum Master, Developer, Designer

01.2021 - 06.2021

VR Anatomy Viewer is a virtual reality application in which users may interact with and learn about human anatomy. It is intended to benefit students, teachers, and interested individuals alike. With this first installment of the application, the team's focus was on the heart, as that was our client's expertise.

This project was based in Unity. The final product was a Virtual Reality application allowing users to interact with an anatomically correct human heart to learn more about it.

My role on the project was primarily as scrum master, but I was also responsible for developing key features, as well as designing the overall look and feel of the UI. You can see examples of my work on the project on my LinkedIn.

Story Tracker | Developer, Designer

10.2020 - 12.2020

Story Tracker is web application where writers of all kinds may track their daily progress on writing projects. Users can categorize their entries by project, and they are able to see statistics based on these entries.

This project was a single-page application created using Angular on the front end with a nodeJS back-end. We used MongoDB to track user entries.

On this project, I was one of a three-person development team. I was responsible for such features as the calendar component, the statistics component, and the login component. This involved ensuring user input was handled correctly, and that data displayed correctly as well.
